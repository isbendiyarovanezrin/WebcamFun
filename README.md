## Photo booth 📸

##### _"Photo booth" project built with HTML5, CSS3 and vanilla JavaScript._

##### _You can view the project [here](https://isbendiyarovanezrin.github.io/WebcamFun "Click me!😎")._

## Installation and use

##### _Install npm packages:_

```
npm install
```

##### _Start working:_

```
npm start
```
